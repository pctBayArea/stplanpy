stplanpy.acs module¶

The functions in this module can be used to import American community survey (ACS) 2011-2015 origin destination data into Pandas.

stplanpy.acs.clean_acs(fd: pandas.core.frame.DataFrame, returns=False, groups=True, home=True, reduced=True, error=False) → pandas.core.frame.DataFrame¶

Test

stplanpy.acs.read_acs(file_name)¶

Import ACS origin destination data

This function imports ACS 2011-2015 origin destination (OD) data into a Pandas dataframe. In the output dataframe there is one row per origin destination pair. The column names and their ACS definitions are shown in the table below. For each column name there is an additional column with the margin of error. E.g. in addition to “all” there is a column “all_error”.

The origin destination flow data can be found on the website of the American Association of State Highway and Transportation Officials (AASHTO) through their Census Transportation Planning Products Program (CTPP).

Parameters

file_name (str) – File_name is the name and path of an ACS csv file

Returns

Return type

pandas.DataFrame

See also

Examples

The example data file, “od_data.csv”, can he found on github in the examples folder.

from stplanpy import acs

flow_data = acs.read_acs("od_data.csv")
